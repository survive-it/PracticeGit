vue와 git연습용

실행방법 
터미널열고 npm install -> npm run dev

1. 목표: 네이버 클론코딩
2. 환경: 1) front: vue.js, vuex, scss 기반  2) back: node.js 기반
3. 구현순서 1) 네이버 화면 적응형 퍼블리싱 2) 네이버 화면 기능(스와이퍼, 모달창 등) 3) node이용 로그인구현
